สูตรจำเป็นภาคตัดกรวย ม.4

1. วงกลม: (x - h)² + (y - k)² = r²
2. จุดศูนย์กลาง (h, k), รัศมี r
3. พาราโบลา: (y - k)² = 4c(x - h) (แนวนอน)
4. จุดยอด (h, k), โฟกัส (h+c, k), ไดเรกตริกซ์ x = h - c
5. พาราโบลา: (x - h)² = 4c(y - k) (แนวตั้ง)
6. วงรี: (x-h)²/a² + (y-k)²/b² = 1 (a≥b)
7. จุดศูนย์กลาง (h,k), แกนเอกยาว 2a, แกนโทยาว 2b
8. c² = a² - b² (วงรีตามแกน X)
9. ไฮเพอร์โบลา: (x-h)²/a² - (y-k)²/b² = 1
10. c² = a² + b², โฟกัส (h±c, k)
11. ไฮเพอร์โบลา: (y-k)²/a² - (x-h)²/b² = 1
12. เส้นกำกับ: y-k = ±(b/a)(x-h)
ข้อควรระวัง: วงกลมต้อง r > 0, วงรี a ≥ b > 0, ไฮเพอร์โบลา a,b > 0
